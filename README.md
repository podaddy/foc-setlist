# FOC Setlist

🦪 **Fried Oyster Cult** practice setlist with YouTube links.

Live at: **https://podaddy.github.io/foc-setlist/** *(after setup)*

## How it works

1. Drop a setlist image in the Google Drive folder
2. OCR extracts the song names
3. Script matches songs to the band's Google Sheet (with YouTube links)
4. Generates this static page
5. Push to GitHub → auto-published via GitHub Pages

## Regenerate

```bash
cd C:\scripts\oysterbot\setlist_page
python generate_setlist_page.py
```

Then commit and push:

```bash
git add docs/
git commit -m "Update setlist"
git push
```

## Setup (one-time)

1. Create repo `foc-setlist` on GitHub (public)
2. Enable GitHub Pages (Settings → Pages → Source: `main` branch, `/docs` folder)
3. Clone and push this folder
